# Helen Yang Portfolio Case Site

This folder is a static website version of the DS marketing case study.

## Files

- `index.html`: landing page for the portfolio case
- `styles.css`: standalone styling
- `assets/ds-marketing-case-study.pdf`: review version of the case study
- `assets/ds-marketing-case-study.py`: Python version of the analysis
- `assets/ds-marketing-case-study.ipynb`: notebook version

## Fastest way to get a public URL

### Option 1: Netlify Drop

1. Open [Netlify Drop](https://app.netlify.com/drop)
2. Drag the entire `portfolio-site` folder into the browser
3. Netlify will give you an instant public URL

### Option 2: GitHub Pages

1. Create a new public GitHub repo
2. Upload the contents of `portfolio-site`
3. Enable GitHub Pages from the repo settings
4. Your site will be available at a GitHub Pages URL
